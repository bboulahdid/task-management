# Task Management API

A Nest.js based API for task management

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Docker

To run a `PostgreSQL` container :

`docker run --name postgres-db -e POSTGRES_PASSWORD=Top2020 -p 5432:5432 -d postgres`
